Perjalanan Karier

Saya adalah lulusan Teknik Informatika dari Universitas BINUS yang sedang mulai mencari peluang karier di bidang IT.

Minat saya terletak pada pengembangan aplikasi, karena saya sangat menikmati proses merancang dan mengubah desain tersebut menjadi situs web maupun aplikasi mobile yang fungsional. Saya juga memiliki ketertarikan yang kuat pada pengembangan backend, di mana saya berfokus pada pembuatan API dan memastikan komunikasi yang lancar antar komponen sistem.

Saya merasa sangat puas dengan mata kuliah seperti Program Design Methods, di mana saya belajar membuat diagram secara detail sebelum mulai melakukan coding, serta Human-Computer Interaction, di mana saya menggunakan alat seperti Figma untuk merancang dan kemudian mengimplementasikan desain tersebut melalui kode. Selain itu, saya juga menikmati mempelajari Framework Layer Architecture, karena membantu saya memahami dan menerapkan pola MVC (Model-View-Controller) dengan baik.

Kombinasi keterampilan dan minat ini menjadi pendorong semangat saya untuk berkarier di bidang IT, di mana saya berharap dapat memberikan kontribusi yang bermakna dan terus mengeksplorasi dunia teknologi yang terus berkembang.
